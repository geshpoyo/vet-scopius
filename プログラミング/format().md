# format関数とは

  

## 概要

Pythonの`format()`関数は、文字列の中に変数や値を埋め込むための便利な方法です。 

波かっこ `{}` を使って、好きな場所に値を挿入できます。

```

text = "{0}さんは{1}歳、{0}さんの趣味は{2}です。".format("フタバ", 18, "読書")

print(text)

# → フタバさんは18歳、フタバさんの趣味は読書です。

  

text = "{name}さんは{age}歳です。".format(name="フタバ", age=18)

print(text)

# → フタバさんは18歳です。
```



```
num = 3.14159

text = "円周率は{:.2f}です。".format(num)

print(text)

# → 円周率は3.14です。
```

小数点や桁数の指定もできる

- `{}` の中に何も書かない場合、順番に値が入る
- `{0}` `{1}` のように番号を指定すると順番を入れ替えられる
- `{name}` のように名前を付けて指定できる
- `:.2f` のように書式指定で小数点以下の桁数などを調整できる
