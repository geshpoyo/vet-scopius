Just the Docsでは、**Front Matter**で階層を指定します。
```
---

layout: default

title: 動物別疾患

nav_order: 2

has_children: true

---
```

`has_children: true` → 子ページを持つことを宣言


子ページの設定

```
---

layout: default

title: 犬の疾患

parent: 動物別疾患

nav_order: 1

---

``
```
- `parent:` → 親ページ名と一致させる
- `nav_order:` → 並び順を指定

```
---

layout: default
title: 
nav_order: 
nav_exclude: 

published: 
parent: 
has_children: true

category: 
tags: 

---
```
